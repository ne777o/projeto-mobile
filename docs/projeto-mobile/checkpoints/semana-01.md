# Checkpoint 01 - Configuração do Ambiente e Estrutura Inicial

## Data: 15/10/2024
## Status: Concluído ✅

### 1. Preparação do Ambiente
Para iniciar o projeto, foram instaladas as seguintes ferramentas essenciais no Windows:

*   **Node.js (LTS):** Ambiente de execução JavaScript.
*   **Git:** Controle de versão.
*   **VS Code:** Editor de código.
*   **Android Studio:** Para emulação do dispositivo Android (SDK Platform-Tools configurado).
*   **Expo Go:** Instalado no dispositivo físico para testes via QR Code.

### 2. Criação do Projeto
O projeto foi inicializado utilizando o template TypeScript do Expo:
```bash
npx create-expo-app mobile-app -t expo-template-blank-typescript
cd mobile-app
